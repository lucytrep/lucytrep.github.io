# Index by Lucille Trepanier

This is a collection of all of my Type and Interaction 1 and 2 projects throughout the year at Parsons School of Design. The work explores typography, interaction, accessibility, and responsive digital systems.

## Features

- Responsive layout across mobile, tablet, and desktop
- Nav with personal links
- Stretched display typography that scales with viewport width
- Flexible project grid system
- Hover interactions on project thumbnails
- Metadata integration for sharing
- Favicon and Apple touch icon support
  
## Built With

### HTML
- Semantic page structure using `<nav>`, `<main>`, `<ul>`, and `<div>`
- Metadata for accessibility and social sharing
- Responsive design with meta tag
- External font loading through Adobe Fonts / Typekit
- Linked project navigation using anchor tags

### CSS
- Flexbox for navigation and layout structure
- CSS Grid for responsive project layouts
- Media queries for mobile, tablet, and desktop breakpoints
- CSS transforms for stretched typography effects
- Relative units including `rem`, `vw`, and percentages
- Hover transitions and subtle motion interactions
- CSS custom properties (`:root`) for reusable variables
- Responsive typography using `clamp()`
- Overflow handling and viewport scaling adjustments
